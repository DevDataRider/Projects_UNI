# 📊 Projects_UNI

Repositorio que reúne proyectos desarrollados durante mi formación en ingeniería y análisis de datos. Incluye soluciones orientadas a inteligencia de negocios, desarrollo web y procesamiento de datos.

---

## 🚀 Proyectos

### 🔹 Sistema de Tickets OBU

Aplicación web orientada a la gestión de incidencias y tickets.

**Tecnologías:**

* HTML, CSS, JavaScript
* jQuery
* Estructura tipo MVC

**Características:**

* Registro y seguimiento de tickets
* Interfaz interactiva
* Manejo de logs y sesiones

---

### 🔹 Dashboard Analítico (Power BI)

Dashboard enfocado en análisis de datos para la toma de decisiones.

**Tecnologías:**

* Power BI
* Modelado de datos (modelo estrella)
* DAX

**Características:**

* Visualización de KPIs
* Análisis dinámico
* Integración de múltiples fuentes de datos

---

## 🧠 Objetivo

Este repositorio tiene como finalidad demostrar habilidades en:

* Ingeniería de datos
* Análisis y visualización
* Desarrollo de soluciones tecnológicas
